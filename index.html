Face ID

Apple's Face ID is a facial-recognition technology that launched on the iPhone X in 2017. The technology replaces Apple's Touch ID fingerprint scanning system for the company's latest iPhones.

Найчастіше ми зустрічаємо цю технологію, користуючись телефонами, коли їх потрібно розблокувати або розрахуватися за щось. Цікаво дізнатись більше про те, де ще використовується Face ID і які воно має надоліки та переваги? Читайте більше на нашому сайті!

КОРИСНЕ ВІДЕО

Face ID працює за допомогою технології True Depth. Пропонуємо розглянути алгоритм її роботи:

  1. Людина розташовує телефон навпроти свого обличчя так, щоб його повністю бачила камера
  2. Камера True Depth сканує Ваше обличчя інфрачервоним світлом
  3. Точковий проектор наносить на Ваше обличчя малюнок з більш ніж 30 000 невидимих інфрачервоних точок
  4. Камера фіксує унікальну 3D-карту Вашого обличчя і використовує її надалі
  5. Надалі пристрій використовує цю карту для розпізнавання обличчя

Щоб краще ознайомитися з технологією Face ID, радимо переглянути відео презентації цієї функції від Apple.
